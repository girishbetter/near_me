## 🔧 1. Fix Event Link Extraction (HIGH PRIORITY)

### Problem:

* Many `event_url` values are:

  * "#"
  * empty
  * incorrect container links
  * repeated base URLs

### Solution:

Update `/scrapers/unstopScraper.js`:

* Extract ONLY valid `<a>` tags with real `href`
* Ensure:

  * URL starts with `https://`
  * Not equal to "#" or empty
* If multiple anchors exist in card:

  * Prefer anchor wrapping title or main clickable element

### Add validation function:

```javascript

  return url && url.startsWith("https://") && !url.includes("#");
}
```

* Skip events where URL is invalid

## 🔧 3. Remove Existing Dummy Data (DATABASE CLEANUP)

Run cleanup logic:

Delete rows where:

* url is NULL
* url = '#'
* url does not start with "https://"
* title is NULL or empty

SQL:

```sql
DELETE FROM events
WHERE url IS NULL
   OR url = '#'
   OR url NOT LIKE 'https://%'
   OR title IS NULL
```
