## ❌ PROBLEM

Unstop is a heavily dynamic React-based site:

* Content loads via API calls (NOT static HTML)
* Playwright scraping likely failing due to:

  * wrong selectors
  * not waiting for data
  * not triggering API calls

---

## ✅ SOLUTION (MANDATORY)

### STEP 1: USE NETWORK INTERCEPTION (BEST METHOD)

Instead of relying only on DOM scraping:

* Use Playwright to:

  * listen to network requests
  * capture API responses (XHR / fetch)

Example:

* Listen for `/api/search` or similar endpoints
* Extract JSON directly from response

---

### STEP 2: FALLBACK TO DOM SCRAPING

If API interception fails:

* Use stable selectors
* Avoid dynamic class names
* Use anchor tags for links

---

### STEP 3: WAIT STRATEGY (VERY IMPORTANT)

DO NOT use simple timeouts

Use:

* wait_for_load_state("networkidle")
* wait_for_selector() with proper selector

---

### STEP 4: DEBUG MODE

Add:

* screenshot on failure
* console logs of page content length
* log number of elements found

---

### STEP 5: VALIDATION

Ensure scraper returns:

* at least 10+ events
* valid URLs (not "#")
* non-empty titles

If not → log error
