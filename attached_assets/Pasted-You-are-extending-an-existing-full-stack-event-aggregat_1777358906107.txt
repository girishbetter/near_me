# 🔍 2. UNSTOP CRAWLER

File: `/scrapers/unstopCrawler.js`

## Target URL:

https://unstop.com/competitions?oppstatus=open

---

## Scraping Strategy (IMPORTANT)

* Use Playwright (Chromium)
* Wait for full page load
* Handle dynamic content loading

### Infinite Scroll:

* Scroll until:

  * no new content loads OR
  * max 6 scrolls

---

## Data Extraction

From each event card extract:

* title
* event_url (STRICT validation required)
* image
* deadline
* mode
* tags
* organizer

---

## URL VALIDATION (CRITICAL)

Reject:

* "#"
* empty strings
* duplicate URLs

Accept only:

```javascript
url.startsWith("https://unstop.com")
```

---

## Detail Page Enhancement (IMPORTANT)

For higher accuracy:

* Visit each valid event_url (limit concurrency to 3)
* Extract:

  * full description
  * exact dates
  * organizer
  * location
